p
Paragraph tag: you put your paragraph between the open and closed tag you see above.
b
Putting words between the tags above you are bolding the text you put between them.
i
Putting words between the tags above you are italicizing the text you put between them.
sup
The tag above is the superscript tag and you use it for text you need to be affect by the superscript. Example: 22 (power of)
sub
The tag above is the subscript tag and you use it for text you need to be affect by the subscript. Example: H2O
br /
This tag represents a line break. It adds a line break inside or between text.
hr /
This tag represents a horizontal rule. You can add a horizontal rule between sections.
strong
You use these tags to indicate that whatever text you put inside them has strong importance.
em
You use these tags to indicate that the text within them is emphasized. The content within the tags show in italic.
blockquote
This tag is for when an entire paragraph is a quote.
q
The tag above is used for shorter quotes within a paragraph.
abbr
This tag is used for abbreviations and or acronym.
cite
You use this tag to reference a piece of work like a book, film, or research paper.
dfn
You use this tag if you are explaining a new term for the first time.
address
This tag can be used to contain contact information for the author of the page.
ins
This tag shows content that has been added.
del
This tag shows content that has been removed.
s
This tag indicates that something is no longer right or useful.
ol
This tag stands for ordered list. This tag will allow you to create an ordered list within your HTML document.
ul
This tag stands for unordered list. This tag will allow you to create an unordered list within your HTML document.
li
This tag stands for list item. You used this tag to put list items in your ordered list or unordered list.
dl
This tag represents that you are starting a definition list. This tag holds a series of terms and their definitions.
dt
You use this tag to house the term that you will later define.
dd
This tag is used to house the definition of your term.
